### 书源配置一鍵导入方式说明（6.2.0版本之后提供的新功能）
在爱阅书香中选择设置→书源管理→同步→填入项目地址（可输入多个地址）→获取书源配置：

```markup
https://gitee.com/mxyseo/iosread
```
ps：
此方式能保存书源地址，后续需要更新时，只需要点击获取书源配置就可以了，推荐使用。<br/><br/>
