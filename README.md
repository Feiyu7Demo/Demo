# iosread
IOS爱阅书香书源设置等备份  Update:2020年4月14日
## 书源配置导入说明
在爱阅书香中选择设置→书源管理→同步→填入项目地址→获取书源配置:

```markup
https://github.com/small-bao/iosread
```
